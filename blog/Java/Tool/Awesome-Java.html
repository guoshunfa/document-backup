                                                                                                            # Build Tool

                                                                                                            Tools which handle the buildcycle of an application.

                                                                                                            • Apache Maven (opens new window) - Declarative build and dependency management which favors convention over configuration. It's preferable to Apache Ant which uses a rather procedural approach and can be difficult to maintain.
                                                                                                            • Gradle (opens new window) - Incremental builds which are programmed via Groovy instead of declaring XML. Works well with Maven's dependency management and treats Ant scripts as first-class citizens.

                                                                                                            # Bytecode Manipulation

                                                                                                            Libraries to manipulate Java bytecode programmatically.

                                                                                                            • ASM (opens new window) - All purpose, low level, bytecode manipulation and analysis.
                                                                                                            • Byte Buddy (opens new window) - Further simplifies bytecode generation with a fluent API.
                                                                                                            • Javassist (opens new window) - Tries to simplify the editing of bytecode.

                                                                                                            # Cluster Management

                                                                                                            Frameworks which can dynamically manage applications inside of a cluster.

                                                                                                            • Apache Aurora (opens new window) - Apache Aurora is a Mesos framework for long-running services and cron jobs.
                                                                                                            • Singularity (opens new window) - Singularity is a Mesos framework that makes deployment and operations easy. It supports web services, background workers, scheduled jobs, and one-off tasks.

                                                                                                            # Code Analysis

                                                                                                            Tools that provide metrics and quality measurements.

                                                                                                            • Checkstyle (opens new window) - Static analysis of coding conventions and standards.
                                                                                                            • Error Prone (opens new window) - Catches common Java mistakes as compile-time errors.
                                                                                                            • FindBugs (opens new window) - Static analysis of bytecode to find potential bugs.
                                                                                                            • PMD (opens new window) - Source code analysis of bad coding practices.
                                                                                                            • SonarQube (opens new window) - Integrates other analysis components via plugins and provides an overview of the metrics over time.

                                                                                                            # Compiler-compiler

                                                                                                            Frameworks that help to create parsers, interpreters or compilers.

                                                                                                            • ANTLR (opens new window) - Complex full-featured framework for top-down parsing.
                                                                                                            • JavaCC (opens new window) - More specific and slightly easier to learn. Has syntactic lookahead.

                                                                                                            # Configuration

                                                                                                            Libraries that provide external configuration.

                                                                                                            • config (opens new window) - Configuration library for JVM languages.

                                                                                                            # Constraint Satisfaction Problem Solver

                                                                                                            Libraries that help on implementing optimization and satisfiability problems.

                                                                                                            • Choco (opens new window) - An off-the-shelf constraint satisfaction problem solver, which uses constraint programming techniques to solve constraint satisfaction problems.
                                                                                                            • JaCoP (opens new window) - A constraint programming solver that includes an interface for the FlatZinc language (opens new window), enabling it to execute MiniZinc (opens new window) models.
                                                                                                            • OptaPlanner (opens new window) - A business planning and resource scheduling optimization solver.
                                                                                                            • Sat4J (opens new window) - A state-of-the-art SAT solver for boolean and optimization problems in Java.

                                                                                                            # Continuous Integration

                                                                                                            Tools which support continuously building, testing and releasing applications.

                                                                                                            • Bamboo (opens new window) - Atlassian's solution with good integration of other products. You can either apply for an Open Source license or buy it.
                                                                                                            • CircleCI (opens new window) - Hosted service with a free trial.
                                                                                                            • Codeship (opens new window) - Hosted services with a limited free plan.
                                                                                                            • Go (opens new window) - ThoughtWork's CI open source solution.
                                                                                                            • Jenkins (opens new window) - Provides server-based deployment services.
                                                                                                            • TeamCity (opens new window) - JetBrain's CI solution with a free version.
                                                                                                            • Travis (opens new window) - Hosted service often used for open source projects.

                                                                                                            # CSV

                                                                                                            • Frameworks and libraries that simplify reading/writing CSV data.*

                                                                                                            • uniVocity-parsers (opens new window) - One of the fastest and most feature-complete CSV parsers for Java. Also comes with parsers for TSV and fixed width records.

                                                                                                            # Database

                                                                                                            Everything which simplifies interactions with the database.

                                                                                                            • Apache Phoenix (opens new window) - High performance relational database layer over HBase for low latency applications.
                                                                                                            • Flyway (opens new window) - Simple database migration with Java API.
                                                                                                            • H2 (opens new window) - Small SQL Database notable for its in-memory functionality.
                                                                                                            • HikariCP (opens new window) - High performance JDBC connection pool.
                                                                                                            • JDBI (opens new window) - Convenient abstraction of JDBC.
                                                                                                            • jOOQ (opens new window) - Generates typesafe code based on SQL schema.
                                                                                                            • Liquibase (opens new window) - An open source database-independent library for tracking, managing and applying database schema changes.
                                                                                                            • Presto (opens new window) - Distributed SQL query engine for big data.
                                                                                                            • Querydsl (opens new window) - Typesafe unified queries for Java.

                                                                                                            # Data structures

                                                                                                            • Apache Parquet (opens new window) - A columnar storage format based on assembly algorithms from the Dremel paper by Google.

                                                                                                            # Date and Time

                                                                                                            Libraries related to handling date and time.

                                                                                                            • Joda-Time (opens new window) - De facto standard date/time-library before Java 8.
                                                                                                            • Time4J (opens new window) - Advanced date and time library for Java.

                                                                                                            # Dependency Injection

                                                                                                            Libraries that help to realize the Inversion of Control (opens new window) paradigm.

                                                                                                            • Dagger (opens new window) - Compile-time injection framework without reflection, mainly for Android.
                                                                                                            • Guice (opens new window) - Lightweight but powerful framework that completes Dagger.
                                                                                                            • HK2 (opens new window) - A light-weight and dynamic dependency injection framework.

                                                                                                            # Development

                                                                                                            Augmentation of the development process at a fundamental level.

                                                                                                            • ADT4J (opens new window) - Algebraic Data Types for Java (JSR-269 code generator).
                                                                                                            • AspectJ (opens new window) - Seamless aspect-oriented programming extension.
                                                                                                            • Auto (opens new window) - Collection of source code generators.
                                                                                                            • DCEVM (opens new window) - Modification of the JVM that allows unlimited redefinition of loaded classes at runtime.
                                                                                                            • Immutables (opens new window) - Scala-like case classes in standard Java.
                                                                                                            • JRebel (opens new window) - Commercial software that instantly reloads code and configuration changes without redeploys.
                                                                                                            • Lombok (opens new window) - Code-generator which aims to reduce the verbosity of Java.
                                                                                                            • RxJava (opens new window) - Library for composing asynchronous and event-based programs using observable sequences from the JVM.
                                                                                                            • Spring Loaded (opens new window) - Another class reloading agent for the JVM.
                                                                                                            • vert.x (opens new window) - Polyglot event-driven application framework for the JVM.

                                                                                                            # Distributed Applications

                                                                                                            Libraries and frameworks for writing distributed and fault-tolerant applications.

                                                                                                            • Akka (opens new window) - Toolkit and runtime for building concurrent, distributed, and fault tolerant event-driven applications.
                                                                                                            • Apache Storm (opens new window) - Distributed realtime computation system.
                                                                                                            • Apache ZooKeeper (opens new window) - Coordination service with distributed configuration, synchronization, and naming registry for large distributed systems.
                                                                                                            • Hazelcast (opens new window) - Distributed and highly scalable in-memory datagrid.
                                                                                                            • Hystrix (opens new window) - Latency and Fault Tolerance for Distributed Systems.
                                                                                                            • JGroups (opens new window) - Toolkit for reliable messaging and creating clusters whose nodes can send messages to each other.
                                                                                                            • Orbit (opens new window) - Virtual Actors, adding another level of abstraction to traditional Actors.
                                                                                                            • Quasar (opens new window) - Lightweight threads and actors for the JVM.

                                                                                                            # Distributed Databases

                                                                                                            Databases in a distributed system that appear to applications as a single data source.

                                                                                                            • Apache Cassandra (opens new window) - Apache Cassandra is a distributed column-oriented database providing high availability with no single point of failure.
                                                                                                            • Apache HBase (opens new window) - Apache HBase is the Hadoop database, a distributed, scalable, big data store.
                                                                                                            • Infinispan (opens new window) - Distributed and highly concurrent key/value datastore used for caching.

                                                                                                            # Distribution

                                                                                                            Tools which handle the distribution of Java applications in native formats.

                                                                                                            • Bintray (opens new window) - Version control for binaries which handles the publishing. Can also be used with Maven or Gradle and has a free plan for Open Source Software or several business plans.
                                                                                                            • Central Repository (opens new window) - Largest binary component repository available as a free service to the open source community (opens new window). Default used by Apache Maven and available in all other build tools.
                                                                                                            • IzPack (opens new window) - Setup authoring tool for cross-platform deployments.
                                                                                                            • JitPack (opens new window) - Easy to use package repository for GitHub. Builds Maven/Gradle projects on demand and publishes ready-to-use packages.
                                                                                                            • Launch4j (opens new window) - Wraps JARs in lightweight and native Windows executables.
                                                                                                            • Nexus (opens new window) - Binary management with proxy and caching capabilities.
                                                                                                            • packr (opens new window) - Packs your JAR, assets and JVM for native distribution on Windows, Linux and Mac OS X.

                                                                                                            # Document Processing

                                                                                                            Libraries that assist with processing office document formats.

                                                                                                            • Apache POI (opens new window) - Supports OOXML (XLSX, DOCX, PPTX) as well as OLE2 (XLS, DOC or PPT).
                                                                                                            • jOpenDocument (opens new window) - Processes the OpenDocument format.
                                                                                                            • documents4j (opens new window) - Java API for document format conversion using third-party converters such as MS Word.

                                                                                                            # Functional Programming

                                                                                                            Libraries that facilitate functional programming in Java.

                                                                                                            • Functional Java (opens new window) - Implements numerous basic and advanced programming abstractions that assist composition oriented development.
                                                                                                            • Javaslang (opens new window) - Functional component library built for Java 8+ that provides persistent data types and functional control structures.
                                                                                                            • jOOλ (opens new window) - An extension to Java 8 which aims to fix gaps in lambda, providing numerous missing types and a rich set of sequential Stream API additions.

                                                                                                            # Game Development

                                                                                                            Frameworks that support the development of games.

                                                                                                            • jMonkeyEngine (opens new window) - Game engine for modern 3D development.
                                                                                                            • libGDX (opens new window) - All-round cross-platform, high-level framework.
                                                                                                            • LWJGL (opens new window) - Robust framework that abstracts libraries like OpenGL/CL/AL.

                                                                                                            # GUI

                                                                                                            Libraries to create modern graphical user interfaces.

                                                                                                            • JavaFX (opens new window) - The successor of Swing.
                                                                                                            • Scene Builder (opens new window) - Visual layout tool for JavaFX applications.

                                                                                                            # High Performance

                                                                                                            Everything about high performance computation, from collections to specific libraries.

                                                                                                            • Disruptor (opens new window) - Inter-thread messaging library.
                                                                                                            • fastutil (opens new window) - Fast and compact type-specific collections for Java.
                                                                                                            • GS Collections (opens new window) - Collection framework inspired by Smalltalk.
                                                                                                            • Koloboke (opens new window) - Hash sets and hash maps.
                                                                                                            • HPPC (opens new window) - Primitive collections.
                                                                                                            • Javolution (opens new window) - Library for real-time and embedded systems.
                                                                                                            • Reactor (opens new window) - Library for building reactive fast-data applications.
                                                                                                            • Trove (opens new window) - Primitive collections.

                                                                                                            # IDE

                                                                                                            Integrated development environments that try to simplify several aspects of development.

                                                                                                            • Eclipse (opens new window) - Established, open-souce project with support for lots of plugins and languages.
                                                                                                            • IntelliJ IDEA (opens new window) - Supports a lot of JVM languages and provides good options for Android development. The commercial edition targets the enterprise sector.
                                                                                                            • NetBeans (opens new window) - Provides integration for several Java SE and EE features starting with database access and servers to HTML5 and AngularJS.

                                                                                                            # Imagery

                                                                                                            Libraries that assist with the creation, evaluation or manipulation of graphical images.

                                                                                                            • Imgscalr (opens new window) - Imgscalr is an very simple and efficient (hardware accelerated) image-scaling library implemented in pure Java 2D.
                                                                                                            • Picasso (opens new window) - Image downloading and caching library for Android.
                                                                                                            • Thumbnailator (opens new window) - Thumbnailator is a high-quality thumbnail generation library for Java.
                                                                                                            • ZXing (opens new window) - Multi-format 1D/2D barcode image processing library.

                                                                                                            # JSON

                                                                                                            Libraries that simplify JSON processing.

                                                                                                            • Genson (opens new window) - Powerful and easy to use Java and Scala to JSON conversion library.
                                                                                                            • Gson (opens new window) - Serializes Java objects to JSON and vice versa. Good performance with on-the-fly usage.
                                                                                                            • Jackson (opens new window) - Similar to GSON but has performance gains if you need to instantiate the library more often.
                                                                                                            • LoganSquare (opens new window) - JSON parsing and serializing library based on Jackson's streaming API. Outpeforms GSON & Jackson's library.

                                                                                                            # JVM and JDK

                                                                                                            Current implementations of the JVM/JDK.

                                                                                                            • JDK 9 (opens new window) - Early access releases of JDK 9.
                                                                                                            • OpenJDK (opens new window) - Open source implementation.

                                                                                                            # Logging

                                                                                                            Libraries that log the behavior of an application.

                                                                                                            • Apache Log4j 2 (opens new window) - Complete rewrite of the previous version. Now has a powerful plugin and configuration architecture.
                                                                                                            • kibana (opens new window) - Analyzes and visualizes log files.
                                                                                                            • Logback (opens new window) - Founded by the same developer as Log4j and proves to be a robust logging library with interesting configuration options via Groovy.
                                                                                                            • logstash (opens new window) - Tool for managing log files.
                                                                                                            • SLF4J (opens new window) - Abstraction layer which is to be used with an implementation.

                                                                                                            # Machine Learning

                                                                                                            Tools that provide specific statistical algorithms which allow to learn from data.

                                                                                                            • Apache Flink (opens new window) - Fast and reliable large-scale data processing engine.
                                                                                                            • Apache Hadoop (opens new window) - Open-source software framework for storage and large-scale processing of data-sets on clusters of commodity hardware.
                                                                                                            • Apache Mahout (opens new window) - Scalable algorithms focused on collaborative filtering, clustering and classification.
                                                                                                            • Apache Spark (opens new window) - Open-source data analytics cluster computing framework.
                                                                                                            • H2O (opens new window) - Analytics engine for statistics over big data.
                                                                                                            • Weka (opens new window) - Collection of algorithms for data mining tasks ranging from pre-processing to visualization.

                                                                                                            # Messaging

                                                                                                            Tools that help sending messages between clients to ensure protocol independency.

                                                                                                            • Aeron (opens new window) - Efficient reliable unicast and multicast message transport.
                                                                                                            • Apache ActiveMQ (opens new window) - Open-source message broker that implements JMS and converts synchronous to asynchronous communication.
                                                                                                            • Apache Camel (opens new window) - Glues together different transport APIs via Enterprise Integration Patterns.
                                                                                                            • Apache Kafka (opens new window) - High-throughput distributed messaging system.
                                                                                                            • JBoss HornetQ (opens new window) - Clear, concise, modular and made to be embedded.
                                                                                                            • JeroMQ (opens new window) - Pure Java implementation of ZeroMQ.

                                                                                                            # Miscellaneous

                                                                                                            Everything else.

                                                                                                            • Design Patterns (opens new window) - Implementation and explanation of the most common design patterns.
                                                                                                            • fabric8 (opens new window) - Integration platform for Java containers.
                                                                                                            • Jimfs (opens new window) - In-memory file system.
                                                                                                            • Lanterna (opens new window) - Easy console text GUI library similar to curses.
                                                                                                            • LightAdmin (opens new window) - Pluggable CRUD UI library for rapid application development.
                                                                                                            • Metrics (opens new window) - Create your own metrics or add them for supported frameworks, then expose them via JMX or HTTP, or send them to a database.
                                                                                                            • OpenRefine (opens new window) - Tool for working with messy data: cleaning, transforming, extending it with web services and linking it to databases.
                                                                                                            • RoboVM (opens new window) - Commercial framework with a free trial to write native iOS apps in Java.

                                                                                                            # Monitoring

                                                                                                            Tools that monitor applications in production.

                                                                                                            • AppDynamics (opens new window) - Commercial performance monitor.
                                                                                                            • New Relic (opens new window) - Commercial performance monitor.
                                                                                                            • Takipi (opens new window) - Commercial in-production error monitoring and debugging.

                                                                                                            # Native

                                                                                                            For working with platform-specific native libraries.

                                                                                                            • JNA (opens new window) - Work with native libraries without writing JNI. Also provides interfaces to common system libraries.

                                                                                                            # Natural Language Processing

                                                                                                            Libraries that specialize on processing text.

                                                                                                            • Apache OpenNLP (opens new window) - Toolkit for common tasks like tokenization.
                                                                                                            • CoreNLP (opens new window) - Stanford's CoreNLP provides a set of fundamental tools for tasks like tagging, named entity recognition, sentiment analysis and many more.
                                                                                                            • LingPipe (opens new window) - Toolkit for a variety of tasks ranging from POS tagging to sentiment analysis.
                                                                                                            • Mallet (opens new window) - Statistical natural language processing, document classification, clustering, topic modeling and more.

                                                                                                            # Networking

                                                                                                            Libraries for network programming.

                                                                                                            • Async Http Client (opens new window) - Asynchronous HTTP and WebSocket client library.
                                                                                                            • Grizzly (opens new window) - NIO framework. Used as a network layer in Glassfish.
                                                                                                            • Netty (opens new window) - A framework for building high performance network applications.
                                                                                                            • OkHttp (opens new window) - An HTTP+SPDY client for Android and Java applications.
                                                                                                            • Undertow (opens new window) - Web server providing both blocking and non-blocking API’s based on NIO. Used as a network layer in WildFly.

                                                                                                            # ORM
