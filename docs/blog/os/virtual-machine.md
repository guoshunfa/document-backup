#**常见问题**

1、VMware Fusion的windos虚拟机打不开，在启动过程中出现“请检查虚拟机设置中的"启动磁盘"设置.如果您尚未安装操作系统,可从 CD/DVD 设”，怎么解决。

**解答：**虚拟机的话需要先下载系统镜像，然后使用虚拟机设置加载iso镜像。然后进入虚拟机bios设置光盘启动。然后就可以正常的安装系统了。
